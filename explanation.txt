Команда git revert используется для отмены изменений, внесённых конкретным коммитом в Git, при этом сохраняя историю коммитов. В отличие от git reset, который может изменить историю, git revert создаёт новый коммит, который эффективно "отменяет" изменения, внесённые указанным коммитом. Вот как это работает: Вместо того чтобы удалять коммит, git revert генерирует новый коммит, содержащий обратные изменения для указанного коммита. Это означает, что если оригинальный коммит добавил строку кода, коммит-отмена удалит эту строку.Вместо того чтобы удалять коммит, git revert генерирует новый коммит, содержащий обратные изменения для указанного коммита.
